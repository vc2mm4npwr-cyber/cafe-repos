# Cafe de Repos - Landing Page

## 概要
「日々に、やさしい休息を。」をコンセプトに制作した架空カフェのランディングページです。

静かな光と余白を活かし、訪れた人がほっと落ち着けるような世界観を意識してデザインしました。
スクロールアニメーションやカードUIを取り入れ、自然で上品な動きを実装しています。

## 実装内容

### ヒーローセクション
- 背景画像のポジション調整
- 画面高さを調整（85vh）
- テキストの可読性を意識したカラー設計

### スクロールアニメーション
- Intersection Observer を用いた fade-up 表示
- 1回のみ発火する上品な仕様

### Menuセクション
- Gridレイアウト
- ホバー時の浮き上がり演出
- 控えめなシャドウによる立体感

### デザイン設計
- ベージュトーンで統一
- 余白を活かしたレイアウト
- 過度な演出を避けた落ち着いた動き
